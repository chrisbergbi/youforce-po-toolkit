# Roadmap Youforce — Voorjaar 2026

Noem nooit items uit andere producten dan het product waar het idee over gaat.

## HR Core

## Onboarding

## Connectiviteit
- In ontwikkeling (nu): Enterprise API — 'all-you-can-eat' op basis van Domein API-data, gericht op eigen IT-landschap
- In ontwikkeling (nu): vacature-integratie via recruitment API vanuit Workflows
- Beschikbaar: Domein API's (AVG-compliant, per domein)
- Beschikbaar: Reporting Data API (dagelijkse actualisatie voor datawarehouse)
- Uitfasering: HR Datafeed stopt per 1 juni 2027, vervangen door Data API & Domein API's
